Connect to the cluster
Steps:

add your key in /private-key  # I used terraform-key-pair.pem (check bastion.auto.tfvars)

aws eks update-kubeconfig --region us-west-2 --name busyqa-eksproject

make sure you run the bash script ebsdriver_and_ingress.sh

point the domain name to the lb in route 53

Delete the lb and the sg of the lb when done, do this before terraform destroy
